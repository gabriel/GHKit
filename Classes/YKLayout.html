Overview

Layout manager, default.

If setting the layout, your view should implement the YKUIViewLayout protocol.

addSubview:

Add subview.

- (void)addSubview:(UIView *)view

Discussion

Call YKLayout#drawSubviews in the superview drawRect: method to draw YKLayout managed subviews.

Subviews managed by the layout don’t automatically handle re-display, or touches. Layout managed subviews are intended for simple views or views in table view cells that need better scrolling performance. For resizing you may need to adjust the contentMode.
