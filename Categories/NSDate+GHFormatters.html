NSDate(GHFormatters) Category Reference

Declared in GHNSDate+Formatters.h
GHNSDate+Formatters.m

Overview

Date parsers, formatting and formatters for ISO8601, RFC822, HTTP (RFC1123, RFC850, asctime) and since epoch.

Tasks

  • + gh_parseISO8601:

    Parse ISO8601 date. For example, “2010-10-07T04:25Z”.
  • + gh_parseRFC822:

    Parse RFC822 encoded date. For example, ‘Wed, 01 Mar 2006 12:00:00 -0400’.
  • + gh_parseHTTP:

    Parse http date, currently only handles RFC1123 date.
  • + gh_parseTimeSinceEpoch:

timeZone

NSDate or nil if timeSinceEpoch was nil
