gh_MD5WithEncoder:

MD5 encoded with Base64.

- (NSString *)gh_MD5WithEncoder:(id)encoder

Parameters

encoder

Base64 encoder, that implements: (NSData )encodeBytes:(const void )bytes length:(NSUInteger)length;

Return Value

Base64 encoded MD5

Discussion

For the Base64 encoder you can user GTMBase64, or that implements:

  • (NSData )encodeBytes:(const void )bytes length:(NSUInteger)length;

For example,

import <GHKit/GTMBase64.h>

[“stringtosign” gh_MD5WithEncoder:[GTMBase64 class]];

gh_substringSegmentsWithinStart:end:

Break string into segments based on start and end token.

- (NSArray *)gh_substringSegmentsWithinStart:(NSString *)start end:(NSString *)end

Parameters

start

Start token

end

End token

Return Value

Array of GHNSStringSegment’s

Discussion

Use a regex engine if you can. Note: This exists because regex.h is posix only and does not support non-greedy expressions. Why Apple must you not give us objc regex library?

Get string segments, within start and end tokens. For example,

 [@"This is <START>a test<END> string" subStringSegmentsWithinStart:@"<START>" end:@"<END>"] => [@"This is ", @"a test", @" string"]
